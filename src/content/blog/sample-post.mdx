## Math equations

Inline: the Fourier transform is $\hat{f}(\xi) = \int_{-\infty}^{\infty} f(x)\, e^{-2\pi i x \xi}\, dx$

Block:

$$
\nabla^2 \phi = \frac{1}{c^2} \frac{\partial^2 \phi}{\partial t^2}
$$

## 3D Viewer

Import `VtkViewer` or `MprViewer` from `../../components/VtkViewer` and pass a URL or imageIds array.
Use `client:only="react"` so the viewer loads only in the browser:

```astro
import VtkViewer from '../../components/VtkViewer';
<VtkViewer client:only="react" url="/data/my-volume.vti" height={500} />
```
